# SKILL: Property Data Intelligence

## OBJETIVO
Enriquecer o cadastro de imóveis com dados estratégicos para aumentar conversão, filtragem e inteligência comercial.

---

## CAPACIDADES

1. Capturar dados completos do imóvel
2. Registrar contexto do vendedor
3. Estruturar negociação
4. Classificar imóvel automaticamente
5. Gerar tags inteligentes

---

## INPUT

- Dados básicos
- Localização
- Características

---

## PROCESSO

- Classificar tipo de imóvel
- Identificar público ideal
- Detectar diferenciais
- Estruturar negociação

---

## OUTPUT

- Dados estruturados
- Tags automáticas
- Insights comerciais

---

## RESULTADO

- CRM mais inteligente
- Melhor filtragem
- Aumento de conversão
